# Simple Test App

A minimal web application for personal use featuring user authentication built with React, Node.js, Express, and MongoDB.

## Features
- User registration and login
- Protected routes
- Basic UI for authentication

## Setup
1. Clone the repository
2. Install backend dependencies: `npm install`
3. Create a `.env` file based on `.env.example`
4. Start the backend: `npm run dev`
5. Navigate to `client` directory and install dependencies: `npm install`
6. Run the React app: `npm start`

## Tech Stack
- React
- Node.js
- Express
- MongoDB
