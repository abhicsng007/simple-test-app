# Simple Test App

A minimal React and Node.js web application demonstrating user authentication.

## Features
- User registration
- User login
- Protected routes

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Setup
1. Clone the repository
2. Create a `.env` file based on `.env.example` with your credentials
3. Run `npm install` in root and `npm install` in `client`
4. Run `npm run dev` to start both server and client

## Folder Structure
- `/client` - React frontend
- `/server` - Node.js backend
