# Simple Test App

A minimal web application demonstrating user authentication using React, Node.js, Express, MongoDB, and JWT.

## Features
- User registration
- User login
- Protected routes

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## Setup
1. Clone the repository
2. Create a `.env` file based on `.env.example` with your MongoDB URI and JWT secret
3. Run `npm install` in root and `npm install` inside `client`
4. Run `npm run dev` to start both server and client concurrently

## Usage
Open `http://localhost:3000` in your browser.
